BUSIEST PASSENGER AIRPORTS PROJECT - PLANNING DOCUMENT

PROJECT OVERVIEW

Analyze and compare passenger traffic patterns across six major global airports (2020-2025). Document the recovery from pandemic disruptions and identify trends in passenger volume patterns.

SMALLER GOALS BREAKDOWN

GOAL 1: DATA VALIDATION AND VERIFICATION
Verify all airport data is accurate, identify missing values, and check data types are correct.

GOAL 2: EXPLORATORY DATA ANALYSIS
Calculate summary statistics, ranking, and growth rates; identify key patterns (recovery timing, volatility).

GOAL 3: CREATE TABLE VISUALIZATION
Format and style a table with years as rows, airports as columns, and summary metrics (average, total, growth).

GOAL 4: CREATE LINE PLOT VISUALIZATION
Create multi-line plot showing passenger trends over time with color-coded airports and clear legends.

GOAL 5: EXTRACT KEY INSIGHTS
Identify which airport recovered fastest, had strongest growth, and what patterns reveal about global travel.

GOAL 6: CREATE ALTERNATIVE VISUALIZATION
Optional: Create a different chart type (faceted plots, bar chart, heatmap) to highlight different patterns.

GOAL 7: DOCUMENT FINDINGS IN REPORT FORMAT
Write a narrative summary explaining visualizations, insights, data sources, and conclusions.

GOAL 8: CODE ORGANIZATION AND DOCUMENTATION
Organize R script with clear sections, comments, and documentation for reproducibility.

NOTES AND CONSIDERATIONS

- Beijing Daxing (PKX) has missing 2020 and 2022 data (opened in 2019, still ramping up)
- 2020 represents pandemic year with dramatically reduced travel
- 2021-2022 show recovery period
- 2023-2025 shows normalized/mature traffic levels
- Different airports serve different regions/markets (may explain variation)
- Seasonal patterns not visible in annual data (could discuss as limitation)
